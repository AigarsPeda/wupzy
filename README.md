# Game Tracker

### Requirements

  - [Node.js](https://nodejs.org/en/)
  - [Yarn](https://yarnpkg.com/)

### Install dependencies

    yarn install

### Create table

    yarn prisma db push

### Generate Prisma Client

    yarn prisma generate

### Run the app

    yarn start

### Run the app in development mode
  
    yarn dev
